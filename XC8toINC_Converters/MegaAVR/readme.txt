This tool is a converter from MPLAB to dat fot ATTINY and ATMEGA

Run the exe with
  1. No parameter to list all the parts supported, version etc.
  2. Parameter is the chip ie ATMEGA328PB or MEGA328PB

Pipe the out to a file for further use.

If the format of the source files is not compatible the exe will complain with "I/O REGISTER DEFINITIONS text not found" - redevelop a new solution for these chips....or, more likley this is not supported by GCB.
